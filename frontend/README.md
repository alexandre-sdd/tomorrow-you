# Frontend

Frontend app for onboarding, self selection, conversation, and debrief flows.

## Responsibilities
- Render interview and profile reveal UX
- Display future-self options
- Run conversation UI (voice/text, subtitles, controls)
- Show debrief/summary outputs

## Main Areas
- `frontend/screens/` for route-level views
- `frontend/components/` for reusable UI blocks
- `frontend/lib/` for API/client/state utilities

## Backend Dependency
Consumes backend onboarding, pipeline, conversation, and future-self endpoints.
